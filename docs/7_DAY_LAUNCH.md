## Emergency Playbook

**If something breaks:**
1. Acknowledge in Telegram immediately
2. Fix or communicate ETA
3. Follow up when fixed

**If negative feedback:**
1. Thank them for feedback
2. Ask clarifying questions
3. Fix if valid, explain if misunderstanding

**If a post goes viral:**
1. Pin the post
2. Add CTA in replies
3. Engage with every comment
4. Prepare server for traffic spike
